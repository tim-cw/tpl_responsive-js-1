Les boutons

  • À l'aide de la propriété css cursor, assurez-vous que la main apparaisse au survol. (La propriété ne va PAS dans la pseudo-classe :hover)
  • Dans le HTML, ajoutez la classe css préfixée js-btn-local sur les trois boutons
  • Dans le CSS de votre bouton, en utilisant l'imbrication (et l'éperluette), ajoutez un sélecteur pour la classe css active avec les propriétés suivantes (utilisez les variables) :

    • Couleur de bordure (border-color seulement) en noir

Les classes d'état

C'est un concept utilisé pour définir des styles spécifiques qui ne doivent s'appliquer uniquement que lorsqu'un élément HTML se trouve dans un certain état ou un contexte particulier.

Elles sont ajoutées / enlevées avec une interaction de l'usager. Vous utilisez certaines classes d'état depuis longtemps, les pseudos-classes comme :hover par exemple sont des classes d'état également.

Il est possible de créer les nôtres avec un peu de JavaScript et du css comme avec la classe css active du numéro précédent.

Parfois, on veut qu'une classe d'état n'affecte qu'un seul élément, on ajoute alors la dite classe sur l'élément qui doit changer d'état. C'est une classe d'état locale.

On peut aussi avoir besoin d'une classe d'état qui pourrait effecter plusieurs éléments. On la met alors sur le plus haut élément HTML possible afin de pouvoir l'utiliser dans les différents sélecteurs de ses enfants. C'est alors une classe d'état globale.

Liste des éléments permis en JavaScript

  • querySelector (il n'y aura qu'un bouton) avec une classe css préfixée
  • addEventListener
  • event.currentTarget
  • le toggle de classList

À vous de jouer

Pour la portion JavaScript, voici ce que vous devrez faire en utilisant exclusivement les éléments de la liste ci-haut.

  • Ajouter la classe css préfixée js-btn-global au bouton
  • Créez une méthode nommée initGlobalState et appelez là à la fin de votre méthode init. La logique de ce numéro démarre dans cette méthode.
  • Au clic du bouton, vous devez appeller la méthode toggleGlobal et faire alterner :
    • la classe css bigger sur la balise <html> afin de changer son état.
    • la classe css active sur le bouton cliqué afin de changer son état comme au numéro précédent.
  • Utilisez l'inspecteur d'élément pour voir que les classes alternent bien au clic sur les 2 éléments.

Faisons maintenant grossir des éléments

  • Dans headings.scss, à la fin, créez une règle débutant par la classe d'état .bigger.
  • À l'intérieur de cette règle, nous allons changer la taille de quelques éléments de texte à l'aide de la méthode css calc(). Cette méthode permet d'effectuer des opérations mathématique de base sur des valeurs.
    • h1: On va doubler la taille du font-size avec la valeur suivante: calc(var(--font-size-h1) * 2). L'espace entre le multiplié et le 2 est essentielle pour que ça fonctionne.
    • h2: Mutipliez sa valeur actuelle par 1.5
    • .button: Changez simplement sa valeur pour 2rem
  • Votre bouton devrait maintenant avoir un état actif lorsque cliqué et les titres et le bouton devraient grossir considérablement.

Les attributs data

Les éléments HTML peuvent contenir plusieurs attributs qui répondent à différents besoins (class="" pour pouvoir y lier des styles, src="" pour y mettre un chemin vers une image, href="" pour indiquer où doit se rendre le navigateur sur un clic par exemple.

Ces attributs font parti nativement du HTML et ont des rôles bien précis. Il est possible de créer des attributs personnalisés et de pouvoir les utiliser en JavaScript par la suite pour aller y chercher la valeur. Ces attributs sont nommés attributs data.

On les nomme ainsi car ils doivent impérativement débuter par data- suivi du nom de l'attribut. Par exemple: data-user="JF Leblanc" ou encore data-user-email="jf@domaine.com".

Liste des éléments permis en JavaScript

  • querySelectorAll avec des classes css préfixées
  • addEventListener
  • boucle for (celle qui utilise la variable « i »)
  • event.currentTarget
  • la propriété innerText d'un élément HTML pour y changer sa valeur de texte
  • const
  • querySelector pour aller chercher le conteneur dans lequel le texte sera mis à jour
  • concaténation du type "salut, je suis " + variable

À vous de jouer

Pour la portion JavaScript, voici ce que vous devrez faire en utilisant exclusivement les éléments de la liste ci-haut.

  • Ajouter la classe css préfixée js-data aux boutons
  • Créez une méthode nommée initData et appelez là à la fin de votre méthode init. La logique de ce numéro démarre dans cette méthode.
  • Au clic des boutons, vous devez appeller la méthode updateContent. À vous de déterminer si vous devrez utiliser querySelector ou querySelectorAll.
  • Mettez un console.log dans cette nouvelle méthode est testez pour vous assurez que les 3 boutons fonctionnent.

Attribut data

Au clic sur un bouton, il faut aller chercher la valeur contenu dans son attribut data nommé data-name. Tout élément HTML a une propriété nommée dataset qui permet d'accéder à tous les attributs data sur le dit élément.

Concrètement, on veut prendre la valeur du data-name et l'afficher dans une phrase à l'intérieur de l'élément qui contient la classe css js-choice

Pour accéder à la valeur de l'attribut data-name, on irait comme ceci: element.dataset.name. La propriété dataset nous permet d'accéder directement à tout ce qui vient après data-. En résumé dataset.name nous permet d'accéder à la valeur de l'attribut data-name sous la forme d'une chaîne de caractères.

Travaillons dans la méthode updateContent pour les prochaines étapes.

  1. Créez une constante name et assignez-lui la valeur de l'attribut data-name en utilisant la propriété dataset. Dans ce cas-ci, comme on veut aller chercher la valeur sur l'élément cliqué, il faudrait y aller ainsi : event.currentTarget.dataset.name
  2. Testez la valeur dans un console.log. Ce dernier devrait vous retourner la chaîne de caractères contenu dans l'attribut data. Sinon, n'allez pas plus loin et revérifiez votre code.
  3. Assignez à une constante nommée choice une référence de l'élément HTML avec la classe css js-choice. À vous de déterminer si vous devrez utiliser querySelector ou querySelectorAll.. Il n'y en a qu'un seul ici...
  4. Faites un console.log de cette variable et vous devriez voir l'élément HTML dans la console.
  5. Changez la valeur innerText de votre élément HTML en utilisant la variable précédemment créée à l'aide d'une concaténation.
  6. La valeur du innerText devrait ressembler à ceci : "Mon prof préféré est : " + name. name étant la valeur de votre attribut data-name assigné plus haut.

Classe d'état (encore)

Maintenant, afin de d'indiquer quel bouton correspond au text affiché dans le conteneur de choix, il serait intéressant que le bouton ait son état actif. Attention, nous ne voulons pas que les autres boutons soient actifs, uniquement que celui cliqué. Il faudra donc désactiver les autres boutons mais conserver celui cliqué actif.

Je vous explique la logique et c'est à vous de réaliser ce défi.

  1. Au tout début de updateContent, on veut boucler dans tous les éléments qui ont la classe css js-data et s'assurer que la classe active soit retirée.
  2. Ensuite, après la boucle, on veut ajouter la classe active sur l'élément cliqué
  3. Finalement, le code réalisé préalablement pour aller chercher la valeur de l'attribut data-name fera son oeuvre.
